# 知识图谱企业级快速入门与大模型智能体实战

这是一门围绕“知识图谱是什么、为什么需要、如何构建、如何服务大模型与智能体”的系统课程。课程从基础术语出发，逐步进入知识抽取、图数据库、工程流水线、图谱增强检索、智能体知识库和企业级治理。

## 学习目标

- 从历史和动机理解知识图谱，不把它误解成“画图”或“关系型数据库换皮”。
- 掌握实体、关系、路径、多跳、三元组、资源描述框架、本体语言、本体、属性图和图数据库查询语言。
- 理解传统 NLP、深度学习、LLM 三代知识抽取路线。
- 能设计通用型知识图谱，并根据领域需求微调 schema。
- 能解释图谱增强检索、社区检测、近邻关系、实体去重、孤立节点清理等后处理。
- 能把知识图谱作为大模型与智能体的可追踪知识库、检索层和工具层。
- 能识别企业落地阻碍：增量更新、删除级联、schema 漂移、权限继承、多跳路径爆炸、评估监控和成本治理。

## 闭环审核结论

课程已形成“来龙去脉 -> 构建方法 -> 现代作用 -> 工程阻塞点 -> 前沿进展 -> 结课实践”的闭环。代码案例主要集中在第 03-13、15 课；第 01、02、14 课分别是导论、术语和趋势课，不强行安排独立代码。第 05、07、09、10、11、12、15 课已补充课堂内嵌案例，用来连接正文讲解和 `code/` 目录中的工程示例。

## 推荐学习方式
