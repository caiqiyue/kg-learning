# 知识图谱企业级快速入门与大模型智能体实战

这是一门围绕“知识图谱是什么、为什么需要、如何构建、如何服务大模型与智能体”的系统课程。课程从基础术语出发，逐步进入知识抽取、图数据库、工程流水线、图谱增强检索、智能体知识库和企业级治理。

## 学习目标

- 从历史和动机理解知识图谱，不把它误解成“画图”或“关系型数据库换皮”。
- 掌握实体、关系、路径、多跳、三元组、资源描述框架、本体语言、本体、属性图和图数据库查询语言。
- 理解传统 NLP、深度学习、LLM 三代知识抽取路线。
- 能设计通用型知识图谱，并根据领域需求微调 schema。
- 能解释图谱增强检索、社区检测、近邻关系、实体去重、孤立节点清理等后处理。
- 能把知识图谱作为大模型与智能体的可追踪知识库、检索层和工具层。

## 课程结构

| 课时 | 主题 | 重点 |
|---|---|---|
| 01 | 知识图谱的起源 | 从字符串到事物、语义搜索 |
| 02 | 知识图谱基础术语与多跳推理 | 实体、关系、路径、多跳、社区 |
| 03 | 从各种图到知识图谱 | 普通图、属性图、语义图、异构图 |
| 04 | 资源描述框架、本体语言与三元组 | 标准化语义表达 |
| 05 | 信息抽取 | 命名实体识别、关系抽取、实体链接 |
| 06 | 大模型知识抽取 | 结构化输出、schema 约束 |
| 07 | 通用型知识图谱设计 | 文档、分块、实体、社区 |
| 08 | 图数据库与查询语言增删改查 | 增删改查、索引、约束 |
| 09 | 文档到知识图谱工程流水线 | 企业级构建流程 |
| 10 | 后处理 | 近邻、去重、孤立节点、社区检测 |
| 11 | 图谱增强检索与问答 | 局部、全局、混合检索 |
| 12 | 知识图谱如何帮助大模型与智能体 | 可追踪知识库与工具调用 |
| 13 | 企业实践与结课项目 | 架构、评估、治理、上线 |
| 14 | 拓展：知识图谱最新进展与应用 | 图谱增强检索产品化、智能体、虚拟图 |
| 15 | 拓展：企业级工程阻碍与成熟方案 | 增量、删除、权限、成本、成熟产品 |

## 推荐学习方式

先顺序读完 01-04，建立术语、图模型和标准表达基础；再学习 05-09，掌握抽取、schema、图数据库和工程流水线；随后学习 10-13，把后处理、图谱增强检索、智能体和结课项目连成实战主线；最后阅读 14-15 作为拓展课程，了解最新进展和企业级落地阻碍。
