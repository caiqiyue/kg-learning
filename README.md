# 知识图谱企业级快速入门与 LLM/Agent 实战

这是一门围绕“知识图谱是什么、为什么需要、如何构建、如何服务 LLM 与 Agent”的系统课程。课程借鉴本项目 `llm-graph-builder` 的真实工程结构：文档输入、chunk、LLM 抽取、Neo4j 写入、后处理、图谱增强检索 问答和前端可视化。

## 学习目标

- 从历史和动机理解知识图谱，不把它误解成“画图”或“关系型数据库换皮”。
- 掌握实体、关系、三元组、RDF、OWL、本体、属性图和 Neo4j/Cypher。
- 理解传统 NLP、深度学习、LLM 三代知识抽取路线。
- 能设计通用型知识图谱，并根据领域需求微调 schema。
- 能解释 图谱增强检索、社区检测、KNN、实体去重、孤立节点清理等后处理。
- 能把知识图谱作为 LLM/Agent 的可追踪知识库、检索层和工具层。
