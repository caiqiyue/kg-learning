# 联网参考资料

本课程构建时参考了以下公开资料：

- Google Search Blog: Introducing the Knowledge Graph: things, not strings, 2012-05-16.
- W3C RDF 1.1 Concepts and Abstract Syntax.
- W3C OWL 2 Web Ontology Language Document Overview.
- Neo4j Documentation: Cypher getting started and Cypher core concepts.
- Neo4j Graph Data Science Documentation: K-Nearest Neighbors and community detection related pages.
- Neo4j Labs / Developer Documentation: GraphRAG and Neo4j GraphRAG Python package.
- LangChain documentation: LLMGraphTransformer.
- Microsoft Research / arXiv: From Local to Global: A Graph RAG Approach to Query-Focused Summarization, 2024.
- Microsoft GraphRAG documentation: Local Search, Global Search, DRIFT Search.
- Amazon Bedrock documentation: Knowledge Bases with Amazon Neptune Analytics GraphRAG.
- Stardog Enterprise Knowledge Graph Platform: semantic layer, virtualization, inference, governance.
- Ontotext GraphDB product documentation: enterprise RDF graph database and semantic search.
- LlamaIndex PropertyGraphIndex documentation: property graph indexing, upsert and retrieval workflows.

这些资料主要用于补齐历史、标准、Neo4j/Cypher、图算法、GraphRAG 和 LLM 抽取工具链的现代脉络。
